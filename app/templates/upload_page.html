Convert a list of Excel Events to PDF Calendars and an ICS File

You should have a Microsoft XLSX file that has a list of dates (or dates and times) in Column A and a list of events in Column B. Click the "Browse" button below and locate that file. Press the "Create Zip file of PDF and ICS file" button and you will be prompted to open or save a Zip file. You should save it. In that Zip file will be a PDF file with a monthly calendar for each month that has an event and an ICS file that can be imported into your calendar program (Microsoft Outlook, MacOS Calendar, etc.)

An acceptable format for an input file is, for example: 

	1/10/2020 10:30, Tennis Practice
	1/17/2020 10:30, Tennis Practice
	1/17/2020 19:00, Dinner with Pat
	1/24/2020 10:30, Tennis Practice
Or a Microsoft Excel file that looks like:
1/10/2020 10:30 Tennis Practice
1/17/2020 10:30 Tennis Practice
1/17/2020 19:00 Dinner with Pat
1/24/2020 10:30 Tennis Practice
The event time is optional; if you leave it out the event will be an "all day" event.
